## Notes

All 750 tests pass. Key design decisions:
- Tests avoid CSS-class assertions (not reliable in JSDOM without Tailwind processing); structural/behavioural checks are used instead.
- Week number assertion uses the exact combined string `"Week ${weekNumber}"` to avoid false positives from broad regex matches.
- `cellHeight` is verified via inline `style.height` on the DroppableCell element, which receives it as a prop from both the `renderCell` wrapper path and the `GridCell` path.
