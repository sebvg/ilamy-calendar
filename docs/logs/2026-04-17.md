## Notes

- **Hook design**: stop hook uses mtime comparison against today's dev log instead of session-scoped state — simpler, cross-platform (BSD + GNU `stat`), and catches the case even across session restarts. TDD hook is non-blocking to avoid false-positive lockouts on trivial edits (rename, typo).
- **hiddenDays + daily mode**: the regular (non-resource) week view always respects `hiddenDays`. Resource vertical week view respects it in hourly mode only. Non-contiguous visible days in daily mode would require reworking multi-day event positioning, so it's explicitly documented as a limitation rather than silently broken.
- **Naming convention lesson**: file/component names should be self-describing without needing directory path context. `day-header.tsx` inside `horizontal/` is too generic (shows up as `DayHeader` in IDE tabs, grep results, stack traces with no orientation info). Preferred: `resource-week-horizontal-day-header.tsx` exporting `ResourceWeekHorizontalDayHeader`.
